========================================
  TORAH BOT DASHBOARD - QUICK START
========================================

FIRST TIME SETUP:
1. Double-click "setup.bat"
2. Wait for installation to complete

RUNNING THE APP:
1. Double-click "start-dev.bat"
2. Or run: npm run dev (starts Vite on Windows)
3. Open your browser to: http://localhost:5173/
4. You'll see the login page (default landing page)
5. If you don't have an account, click Sign up and create one
6. After signup you will be redirected back to the login page
7. Enter the same credentials on the login page to access the dashboard

PAGES AVAILABLE:
- Login (Default):  http://localhost:5173/
- Signup:           http://localhost:5173/signup
– Dashboard:        http://localhost:5173/ (after authentication)

FEATURES:
✓ Login is the default landing page
✓ Any credentials are accepted (no validation in demo)
✓ Protected dashboard - must signup/login first
✓ Login page with username/password
✓ Signup page with username/password
✓ Links between login and signup
✓ Beautiful Torah Bot design
✓ Fully responsive
✓ Persistent authentication (stays logged in)
